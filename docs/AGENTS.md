# SurrealMind Agent Guide

This folder holds the modular agent docs for SurrealMind. Start here and jump to the focused pages:

- Tools and usage: `docs/AGENTS/tools.md`
- Architecture and guardrails: `docs/AGENTS/arch.md`
- Setup and quickstart: `docs/AGENTS/setup.md`
- Connections and endpoints: `docs/AGENTS/connections.md`
- Maintenance routines: `docs/AGENTS/maintenance.md`
- TODO and roadmap: `docs/AGENTS/todo.md`, `docs/AGENTS/roadmap.md`

Tool roster includes: `think`, `search`, `remember`, `wander`, `rethink`, `corrections`, `maintain`, `howto`,
and delegation tools `call_gem`, `call_cc`, `call_warp`, `call_vibe`, `call_status`, `call_jobs`, `call_cancel`.
